# RaptorX Dashboard

Production-grade dashboard for the AI-powered contextual access control system.

## Requirements

- Node.js 18+

## Setup

```bash
npm install
```

Create an environment file at `.env.local`:

```bash
NEXT_PUBLIC_API_URL=http://localhost:8000
```

## Development

```bash
npm run dev
```

Open http://localhost:3000 to view the dashboard.

## Production Build

```bash
npm run build
npm run start
```
